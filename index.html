A fun interactive learning game about C#

GO ON AN ADVETURE WHILE LEARNING C#

The story follows a student who is lazy and
struggling to learn a programming language,
which is C#. As a player, you need to go on an adventure
while learning C#. There will be some twists and
challenges to test the user's intelligence. The player will
gain some knowledge about the basics of C#.

FRIENDS YOU MADE
ALONG THE WAY

While going on an adventure, you have friends to help you
learn C#. Fionna and Josh will help you go on an adventure.
It's going to be a fun and happy adventure.
